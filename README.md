## Getting Started

Visit to see the live app https://just-todo-it.netlify.com/

### Prerequisites

`Git & NPM`

### Installation steps

```zsh
# clone the repo
git clone https://github.com/shimphillip/just-todo-it.git

# navigate to the directory
cd just-todo-it

# install packages
npm install or yarn

# start the app
npm run start or yarn start
```

## Running the tests

```zsh
npm run test or yarn test
```
